# 详情面板渲染语法说明

本文档说明设置页右侧 Detail 文本的渲染规则（对应 `web_settings_ui/app.js`）。

## 支持语法

1. 换行  
使用 `/n` 表示换行。

2. 加粗  
使用 `[b]...[/b]`。

3. 颜色标签（自动加粗样式由 CSS 控制）
- 严重警告（红）：`[swarn]...[/swarn]`
- 普通警告（黄）：`[warn]...[/warn]`
- 成功/完成（绿）：`[success]...[/success]`
- 信息（蓝）：`[info]...[/info]`

## 示例

```text
[b]控制模式说明[/b]/n
[info]提示：方向、踏板、换挡可独立开关。[/info]/n
[warn]警告：热键重复可能导致冲突。[/warn]/n
[swarn]严重：未检测到虚拟手柄。[/swarn]/n
[success]完成：当前设置已成功应用。[/success]
```

## 注意事项

1. 不使用任何标签时，纯文本仍可正常显示。  
2. 标签必须成对闭合，例如 `[warn]...[/warn]`。  
3. 输入文本会先做 HTML 转义，再执行标签替换。
